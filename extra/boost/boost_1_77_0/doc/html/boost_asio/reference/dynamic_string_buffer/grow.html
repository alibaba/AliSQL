dynamic_string_buffer::grow

DynamicBuffer_v2: Grow the underlying memory by the specified number of bytes. 

void grow(
    std::size_t n);

Resizes the string to accommodate an additional n bytes at the end.

Exceptions

std::length_error

If size() + n > max_size().
